JavaScript API

Getting started

To use the generator in your own scripts just include the klingonName.js into your projects:

<script src="js/klingonName.js"></script>

Then init a generator for each name you want to generate:

var my_klingon_name_a = new klingonName({
    min_syllables:2,
    max_syllables:3
});

var my_klingon_name_b = new klingonName({
    min_syllables:2,
    max_syllables:3
});

Available Options

  • min_syllables: Default is 2
  • max_syllables: Default is 3
  • seed: Generator seed, default is 666
  • phoneme_weights: Array - List of one or more supported phoneme weights. Possible is male or female, default is []
  • phonemes: Array - Feel free to add your own or modified list of phonemes with different weights. Check out the klingonName.js for the default value.
  • syllables: Array - Feel free to add your own list of syllable configurations. Check out the klingonName.js for the default value.

Availabble Methods

Use public methods to get Data from the object:

var random_name = my_klingon_name.getString();
var random_ipa_name = my_klingon_name.getIpaString();
var descriptions = my_klingon_name.getDescriptions();
  • string getString(bool hyphenation, bool split): Returns the full generated name as string
  • string getIpaString(bool hyphenation, bool split): Returns the full generated ipa string of the name
  • array getDescriptions(): Returns an array of objects containing the letter and the description, for each phoneme of the generated name [{string letter,string description},{...}]
